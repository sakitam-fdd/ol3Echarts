## ol3Echarts | a openlayers extension to echarts

- 丰富的图表类型，完整的交互效果，与gis无缝衔接，赋予你的数据更多意义。
- 得益于echrats的强大功能，借助webgl的大数据量的展现，和更多绚丽的特效，更具视觉冲击。
